## Getting Started

1. Clone the repository

```bash
git clone https://github.com/magnusrodseth/geo-genius.git
cd geo-genius
```

2. Install dependencies

```bash
pnpm install
```

3. Run the development server

```bash
pnpm dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser
